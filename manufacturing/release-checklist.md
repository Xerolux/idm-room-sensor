# Fabrication release checklist

- [ ] Exact BOM parts frozen
- [ ] Manufacturer datasheets archived by URL and revision
- [ ] Footprints checked against land-pattern drawings
- [ ] ERC clean or every exception documented
- [ ] DRC clean
- [ ] Copper current and clearance review
- [ ] Analog-output SPICE/bench validation
- [ ] 3D enclosure interference check
- [ ] Gerber visual inspection
- [ ] BOM/CPL import preview checked at manufacturer
- [ ] Prototype ordered as clearly marked engineering sample
